Take any product. Cleave it into the prompts that built it.

《牛刀》（Cleaver）是一个强大的 Claude Code 命令行技能。
它能读取任何成熟的现代产品，将其逆向工程为可直接用于 AI 编程的「系统级提示词 (Prompts)」。不抄袭像素，只提取底层架构与交互灵魂。
